# PackNGo - India Trip Planner

## ✅ Everything is Working!

### Project Overview
A beautiful, modern web application for planning trips across India with comprehensive features including distance calculation, equipment recommendations, budget estimation, safety tips, and emergency contacts.

## 📁 Files Structure
- **lets-travel.html** - Modern Tailwind CSS interface with complete form and results display
- **travel-script.js** - All JavaScript logic for calculations and functionality
- **travel-style.css** - (Legacy CSS - not used in current version)

## 🎯 Features Implemented

### Form Inputs
✅ **Age Input** - Number input for traveler's age (1-120)
✅ **Budget Slider** - Interactive range slider (₹1,000 - ₹100,000)
✅ **Starting Location** - Dropdown with 15 major Indian cities
✅ **Final Destination** - Dropdown with 15 major Indian cities
✅ **Transport Mode** - Radio buttons (Flight, Train, Bus, Car)
✅ **Trip Duration** - Number input for trip days
✅ **Progress Bar** - Visual indicator showing form completion

### Calculations & Features
✅ **Distance Calculation** - Haversine formula using real city coordinates
✅ **Travel Time Estimation** - Based on transport mode and distance
✅ **Equipment Recommendations** - Age-specific and distance-specific packing lists
✅ **Safety Tips** - Transport-specific and distance-aware recommendations
✅ **Budget Breakdown** - Detailed cost estimation in Indian Rupees
✅ **Rest Stops** - Food stops and petrol pumps on major routes
✅ **Weather Information** - Best time to visit and weather details
✅ **Emergency Contacts** - Indian emergency numbers (Police 100, Ambulance 102, etc.)

### User Interface

✅ **Responsive Layout** - Works on mobile, tablet, and desktop
✅ **Interactive Elements** - Hover effects, animations, smooth transitions
✅ **Summary Cards** - Quick overview of trip details

## 🚀 How to Use

1. **Open the file**: Open `lets-travel.html` in a web browser
2. **Fill the form**:
   - Enter your age
   - Adjust budget using the slider
   - Select starting city
   - Select destination city
   - Choose transport mode
   - Enter trip duration (optional)
3. **Click "Plan My Trip"** to generate results
4. **View Results**: See all recommendations including:
   - Distance and travel time
   - Packing list
   - Route stops
   - Cost breakdown
   - Weather info
   - Safety tips
   - Emergency contacts
5. **Print or Reset**: Use buttons to print the plan or start a new trip

## 🔧 How It Works

### Distance Calculation
Uses the Haversine formula to calculate great-circle distance between cities based on latitude/longitude coordinates.

### Equipment Recommendations
- **Age-based**: Different items for children, teens, adults, and seniors
- **Distance-based**: Additional items for journeys over 300km and 600km

### Cost Estimation
- Transport cost per km (varies by mode)
- Accommodation (₹1500/night)
- Food & meals (₹200/hour)
- Tolls & miscellaneous (15% buffer)

### Safety Tips
- Base safety tips for all travelers
- Transport-specific advice
- Extra precautions for journeys over 800km
